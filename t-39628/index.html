Ubuntu环境下搭建方舟：生存进化服务器

发表于 2020-03-30 更新于 2026-04-10 分类于 游戏 阅读次数：

测试环境

华为云-上海二
4 vCPU (Intel Xeon Gold 6278C)
8 GB Memory
40 GB Disk
5Mbit/s Bandwidth
Ubuntu 18.04 x64

连接到服务器

通过游戏内部搜索

通过游戏内的搜索连接可以很简单地加入服务器，有时在游戏内无法找到，可以使用第二种方式。

通过Steam连接

在Steam左上角依次选择视图-服务器，然后输入服务器ip并进行搜索。搜索到服务器后可以加入收藏夹方便下次使用。
