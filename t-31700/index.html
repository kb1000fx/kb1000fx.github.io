网站重回腾讯云的感想
发布于: 2019-08-26 更新于: 2020-02-10 分类于: 碎碎念 阅读次数:

2019年8月26日，这个小网站兜兜转转，又回到了腾讯云，这个她诞生的地方。

2015年的时候，为了和同学一起联机玩MC，我在腾讯云租下了一台学生版的云服务器。在一起联机游戏之余，我整理了一些游戏攻略和指令，写成静态的html页面，用python开了一个简单的http服务器，供自己平日里在游戏时查阅指令。从MC到饥荒再到叛乱，再到各种架设过的游戏，这些攻略与指南和我一起度过了大一大二时当“辣鸡房管”的日子。

后来，因为学习上的事情需要用到Windows Server，腾讯云的服务器被我重装成了Windows Server 2012，这个小网站就在DigitalOcean、CenTex等各大廉价Linux服务商间流浪。在这期间，小站也迎来了一次重大的升级，从几个简单的静态页面变成了WordPress驱动的动态博客，内容上也渐渐加入了一些IT方面的文章。

再后来，腾讯云这面的机器闲置了下来，博客又重新迁了回来，还是 Powered by WordPress 。只不过几个月后，用Nginx替换掉了原先一直在用的Apache。2017年刚回到腾讯云的时候，我就顺手注册了 garybear.cn 这个域名。但是因为那时刚换了新身份证，再加上自己懒癌晚期，就一直没有去备案，域名也一直扔在这吃灰。一年后终于想起来这件事，忙里忙外几趟程序跑下来，这个域名终于正式启用的。在博客之外还开了一个status子域名，在同一台服务器上借助 vhost 开了一个云监控，随时查看各服务器是否宕机。

WordPress是一个很优秀的博客程序，插件丰富，功能强大，可是时间久了也会觉得他过于臃肿，希望换一个简洁、轻量一些的博客。大概是2018年末，我开始关注到了Typecho。之前在WordPress上面我也是通过插件用MarkDown在写文章，所以Typecho的原生支持MarkDown这一点对我还是很有吸引力的。趁着寒假的时间，我把服务器重新拾掇了一遍，服务器环境还是LNMP，还是原来的云监控，只不过这次博客换成了 Powered by Typecho 。寒假的时候正好在家里忙毕设，学习了一下Julia语言，顺便用家里的电脑尝试了一下CUDA GPU计算。其中的一些心得，一些体会，还有毕设项目里的注意点，都让我记录在了博客里。

2019年上半年算是过得比较痛苦吧，半年的时间都在忙毕设，只不过前两个月在家的时候兴致高昂，开学后逐渐咸鱼化，想着已经如此了干脆混个毕业算了。四月份从武汉偷着跑回学校之后心情糟到了极点，直接就把服务器给停了，没备份任何资料直接停的。心里想着反正六月末学生认证就到期了，不如提前和这些东西做个了断。等到快毕业的时候，其实有些后悔，后悔当时太冲动没来得及备份。Julia的资料，毕设项目的整理，还有我做的毕设论文Latex模板的说明，这些东西其实留给下一届的孩子就好了。可惜辛辛苦苦攒齐的这些东西，被我一把全删了。

七月份的时候，我在Github Page上又把小网站搭了起来。这次又变回了最初的静态博客，不过不是简单的小网页了，而是 Powered by Hexo 。之前的大部分文章都找不到了，只是零星的找到了以前的几篇草稿和一些搜索引擎的缓存页面。迁到Github之后又重新提交了一次Google Adsense的申请，之前的服务器因为网络原因一直审核不过，这次居然一次通过了，估计这广告挂着也不会有什么人点，就挂着玩玩了。

网站放在Github上最大的问题就是国内访问速度不稳定，所以打算再加一个国内节点。思考了一下最终还是选择了腾讯云的COS，之前用WordPress和Typecho的时候用COS做过图床，对这个还是比较熟悉的。忙了大半天之后，今天算是正式回来了。

就这样，这个小破站又回到了她最初的起点，她长大了，也变强了。
