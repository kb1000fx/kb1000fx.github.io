左轮手枪中的单动与双动

现代意义上的转轮手枪由美国人塞缪尔·柯尔特(Samuel Colt)发明，因其换弹时弹仓向左甩出常称为左轮手枪。左轮手枪具有结构简单可靠，便于维护等优点，但同时也有气密性差等明显的缺点。
对于左轮手枪和柯尔特有一个很经典的评价：上帝创造了人类，塞缪尔·柯尔特让人人平等(God created men, Samuel Colt made them equal)
左轮手枪按扳机结构可以分为单动和双动两种。其中单动左轮扳机行程短容易击发且精度高，但操作麻烦。双动左轮与其相反，操作简单但由于需要靠扳机的力量完成一系列操作所以会影响到射击精度。

单动左轮 (Single Action Revolvers)

单动式左轮就是电影里西部牛仔常用的那种左轮手枪。
单动式左轮在扣动扳机前，需要用拇指或另一只手来先把击锤扳倒，联动机构会同时让转轮旋转，将下一个弹仓对准击锤和枪管。然后食指扣动扳机，机械装置会释放击锤，让它从扳倒的位置向前运动从而击发子弹。因为扣动扳机只完成了释放击锤这一个动作，所以被称为“单动式”。

图中展示的是德国国防军的第一把制式手枪——M1879式左轮手枪的内部结构，这是一种典型的单动式左轮。

如图所示，每次扣动扳机前都需要手动将击锤扳倒，同时完成换弹的步骤。单动式的扳机行程较短，而且由于只负责释放击锤，所以扳机力度较小，容易击发。

在电影和游戏中常常会出现将扳机扣住，另一只手不断拨动击锤达到速射的场景，如OW和CSGO里的左轮右键。这种方式也被称作 fanning 。当扳机扣住后，原来用于固定击锤位置的棘轮结构就可以自由转动，此时相当于直接操控击锤进行发射，子弹的射速完全取决于手速。

双动左轮 (Double Action Revolvers)

双动左轮与单动左轮不同，在击发过程中不需要手动扳下击锤，而是由扳机带动击锤完成拉下-释放两个步骤。
双动式的扳机有明显的分段感，扳机行程也长于单动。其中第一段为将击锤拉起，即变成单动模式下待击发的状态，同时推动弹仓转动完成换弹。第二段与单动模式中相同，释放击锤击发子弹。

如上图中展示的一样，双动左轮的发射过程能明显区分出两个阶段。CSGO里的左轮扳机分段感也很明显，可以在游戏里体会一下。

现代左轮手枪多采用双动的设计，同时也保留了单动结构。使得左轮在双动射击的同时也可以手动拉下击锤进行单动射击，也可以扣住扳机用手拨动击锤，像单动左轮一样快速射击。

从射击方式上来看，OW里麦克雷使用的的维和者和CSGO里的R8左轮都应该算是兼容单动方式的双动左轮

如图所示，枪机左下角的小凸起其实就是单动中棘轮的变形。如果没有此结构，左轮将只能靠扳机拉下击锤进行双动发射，无法手动拉下击锤进行单动发射。

半自动手枪中的单动与双动

单动半自动手枪 (SA, Single Action Autoloaders)

半自动手枪中单动、双动的概念及其结构都借鉴了左轮手枪的设计。

单动式手枪中最经典的设计就是勃朗宁M1911，我国的54式手枪及其原型TT-34也属于单动式手枪。下面是M1911的结构示意图：

虽然结构相近，但左轮中是靠扳机推动弹仓旋转上弹的，而半自动手枪则是由弹簧来完成这一步骤，所以相同情况下半自动手枪的扳机会更容易扣动。
而且单动左轮中每次射击后都需要手动拉起击锤，而半自动手枪只需要第一次射击时向后拉套筒将子弹上膛，同时将击锤扳倒。
这种区别是由于半自动手枪在发射后火药产生的后坐力会使套筒向后运动，从而自动将击锤扳倒，而不需要每次手动完成此步骤。

下面是M1911发射的过程图：

值得注意的是，对于单动手枪虽然在上膛的时候会扳倒击锤，但并不意味着上膛后就可以随时扣动扳机发射。因为单动式手枪在击锤压下的状态下仍可以将其缓慢放下，此时只有重新将击锤扳倒才能正常使用。
单动式手枪结构简单适合多种环境，多用于军用手枪领域。但由于其使用前需要拉动套筒导致反应过慢，不适合带弹携带等因素在警用领域表现不是很出色。

单动/双动半自动手枪 (DA/SA, Double Action / Single Action)

大部分的双动式半自动手枪确切来说应该叫作单动/双动半自动手枪，即发射第一发子弹时为双动，其余为单动。单动/双动半自动手枪在结构上与双动式左轮类似，第一个采用双动结构的半自动手枪是瓦尔特PP手枪，下面就是其内部结构示意图：

尽管内部结构与双动式左轮的结构十分相似，但双动式左轮每一发射击都需要靠扳机的力量重新将击锤扳倒，即每一次射击时扳机都是双动。
而双动式半自动手枪只有第一发子弹需要靠扳机扳倒击锤，是严格意义上的双动。之后的每一次发射时套筒的后坐力都会重新将击锤扳倒，即第二发及之后的子弹属于单动，扣动扳机只负责击锤的释放。
所以单动/双动半自动手枪在发射时第一发子弹的扳机行程和力度都大于其余的子弹，射击手感也会有区别。

单动/双动式半自动手枪的优点在于综合了单动和纯双动的优点，既可以手动扳倒击锤将第一发变成单动以提高射击精度，也可以在紧急情况下使用双动，只扣动扳机快速进行射击。

纯双动式半自动手枪 (DAO, Double Action Only)

双动式手枪(包括单动/双动和纯双动)不需要在射击前拉动套筒，任何情况下只要膛内有弹就可以扣动扳机进行发射。
与之相比的是单动式手枪即使在子弹上膛时拉下了击锤，也可以缓慢将其归位。
所以双动式的出枪反应速度比单动式更快，配合其他保险装置可以做到膛内带弹，拔枪即用。适合警察等需要应对突发情况的人群。

纯双动与上面的单动/双动区别在于，纯双动在套筒后坐时不会将击锤重新拉下，所以每次扣动扳机都需要将击锤重新拉起后再释放。因此纯双动每次扳机行程都一样，每发子弹的发射手感都相同。同时纯双动手枪多采用隐藏式击锤，也避免了与衣物等钩挂走火的情况。

因为其结构上的特点与警察出枪反应快，带弹携带的需求相吻合，所以目前很多的警用手枪，如P250，都采用纯双动结构。

击针式半自动手枪 (Striker Fired Autoloaders)

格洛克系列手枪是另一种广泛应用的警用手枪。不同于传统的单动或双动式手枪的枪机结构，格洛克系列手枪没有传统意义上的击锤，而是靠一个平行移动的击针来击发子弹，这种结构也被称为 Safe Action 。以下是格洛克手枪的结构示意图：

格洛克手枪的枪机模式介于单动和双动之间。使用前拉动套筒使子弹上膛，同时将击针勾上，随后可以一直带弹携带。击针在运动过程中实际是完成了压缩-释放两个过程，但扳机没有双动明显的段落感。击发后套筒的后坐力带动击针重新勾上。
与传统的单动模式相比，格洛克的扳机实际上带动击针进行了两个动作。
与单动/双动模式相比，每一发子弹的射击方式与手感都完全相同。
与纯双动模式相比，格洛克无法只靠扳机让击针(Striker)回到待击发状态，必须在击针勾上之后才能击发。

格洛克的Safe Action在设计理念上与纯双动比较接近，所以二者都在警用枪械中应用较广。
